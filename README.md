### Prasyarat

Pastikan Anda sudah menginstall **Node.js** (versi 18 atau lebih baru).

#### Install Node.js di Windows:

1. Download dari [nodejs.org](https://nodejs.org/)
2. Pilih versi **LTS** (Long Term Support)
3. Jalankan installer dan ikuti instruksi
4. **Penting:** Pastikan opsi "Add to PATH" tercentang
5. Setelah instalasi selesai, restart terminal/VS Code

#### Verifikasi Instalasi:

```powershell
node -v
npm -v
```

Jika muncul versi (contoh: v20.10.0 dan 10.2.3), instalasi berhasil.

---

### Langkah Instalasi Project

1. **Clone atau download project ini**

2. **Install dependencies:**

```powershell
npm install
```

3. **Jalankan development server:**

```powershell
npm run dev
```

4. **Buka browser:**
   - Akses `http://localhost:5173`
   - Website akan otomatis reload saat ada perubahan file
